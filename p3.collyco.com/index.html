There is no shortage of password generators online. But these generators give you an alphanumeric sequence of letters and numbers that are tough to memorize. This password generator will allow you to select a password first, and then it will make the password more difficult to crack. Input: password
output: P@$$w0rd

Let's go over what happened. The first letter of the word was capitalized to give the password case-sensitivity. all of the a's were changed to @ because that symbol is most closely associated with the letter a. The letter s was converted to the $ symbol. The letter 0 was replaced with a zero. The outputted password will pass most website's difficulty tests and it is still easy to memorize.
